# AI Resume Analyser

🚀 Overview

An AI-powered tool that analyzes resumes and provides feedback based on job descriptions.

🧠 Features

* Resume parsing (PDF/Text)
* Keyword matching with job descriptions
* Score generation
* Suggestions for improvement

🛠 Tech Stack

* Python
* NLP (spaCy / sklearn / etc.)
* Streamlit / Flask (if used)

▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
```

📌 Future Improvements

* Better scoring algorithm
* UI enhancements
* Resume tailoring suggestions
